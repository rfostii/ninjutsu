# 🥷 Ninjutsu

_Those who break the rules are scum... but those who abandon automation are worse._

> Personal macOS dev setup — fast, clean, reproducible.

---

## ⚡ What is this?

**Ninjutsu** is my personal dotfiles setup for quickly bootstrapping a macOS development environment.

One command → and you get:
- a configured shell
- essential tools installed
- Git aliases ready
- Node.js + pnpm set up

---

## 🚀 Quick Start

```bash
git clone <your-repo>
cd Ninjutsu
chmod +x setup.sh
./setup.sh
```

## Examples

```bash
gnew feature/login   # create & switch branch
gcm "fix bug"        # add + commit
mkcd project         # create folder + cd
```
